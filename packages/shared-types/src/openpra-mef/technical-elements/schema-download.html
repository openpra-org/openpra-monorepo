OpenPRA Technical Elements Schema

This JSON Schema defines the structure and validation rules for OpenPRA Technical Elements.

⬇️ Download JSON Schema

Usage Example

// Example of using the schema with a validator
const ajv = new Ajv();
const validate = ajv.compile(schema);
const valid = validate(data);

if (!valid) {
    console.log(validate.errors);
}

For more information, please refer to the OpenPRA documentation.
